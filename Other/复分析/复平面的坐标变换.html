小结

2025/10/19数学例题

问题描述

考虑 zzz 平面上的三角形区域 SSS，其边界由 x=1x = 1x=1, y=1y = 1y=1, y=1−xy = 1 - xy=1−x 给出。对于以下变换，绘制 SSS 在 www 平面上的影像区域 S′S'S′，并给出区域边界的方程：

  1. w=z+(1−3i)w = z + (1 - \sqrt{3}i)w=z+(1−3​i)；
  2. w=2eπi6z+(1−3i)w = 2e^{\frac{\pi i}{6}}z + (1 - \sqrt{3}i)w=2e6πi​z+(1−3​i)；
  3. w=z2w = z^2w=z2。

步骤

将复平面 zzz-平面中的区域 SSS 映射到 www-平面，通常步骤如下：

1. 在 zzz-平面表征区域 SSS：

将区域的边界用方程或参数形式表示。例如本题三角形区域

S:{x=1,y=1,y=1−xS: \quad \begin{cases} x = 1, \\ y = 1, \\ y = 1 - x \end{cases} S:⎩⎨⎧​x=1,y=1,y=1−x​

先确认这三条直线的交点，进而确定三角形三个顶点。

2. 将边界线逐条带入变换 w=f(z)w=f(z)w=f(z)：

  • 当 f(z)f(z)f(z) 是仿射变换（例如 w=az+bw = a z + bw=az+b，其中 a≠0a\neq 0a=0 ）时，直线映射仍然是直线，圆映射仍然是圆或直线；因此可以将边界的直线方程变换到 www-平面得到新直线方程。
  • 当 f(z)f(z)f(z) 是非线性变换（例如 w=z2w = z^2w=z2），需要用参数法（或直接消元）求出映射曲线的明式方程。

3. 找出变换后区域 S′S'S′ 的边界方程：

将上一步得到的三个映射结果写清楚：它们在 www-平面是三条什么曲线（或直线），然后就能圈出对应的三角形或其它形状。

变换类型

一、仿射变换

仿射变换具有形如

w=az+b,a≠0,w = a z + b, \quad a \neq 0, w=az+b,a=0,

的形式。记 z=x+iyz = x + i yz=x+iy，将 www 写成

w=u+iv,w = u + i v, \quad w=u+iv,

则

w=az+b⟹u+iv=a(x+iy)+b.w = a z + b \quad \Longrightarrow\quad u + i v = a(x + i y) + b. w=az+b⟹u+iv=a(x+iy)+b.

如果 a=α+iβa = \alpha + i \betaa=α+iβ，b=γ+iδb = \gamma + i \deltab=γ+iδ，则

az+b=(α+iβ)(x+iy)+(γ+iδ)=(αx−βy+γ)+i(βx+αy+δ).a z + b = (\alpha + i \beta)(x + i y) + (\gamma + i \delta) = (\alpha x - \beta y + \gamma) + i(\beta x + \alpha y + \delta). az+b=(α+iβ)(x+iy)+(γ+iδ)=(αx−βy+γ)+i(βx+αy+δ).

因此

u=αx−βy+γ,v=βx+αy+δ.u = \alpha x - \beta y + \gamma, \quad v = \beta x + \alpha y + \delta. u=αx−βy+γ,v=βx+αy+δ.

在这种变换下：

  • 直线 α0x+β0y+γ0=0\alpha_0 x + \beta_0 y + \gamma_0 = 0α0​x+β0​y+γ0​=0 在 zzz-平面上，会映射成 www-平面上一条仍是直线的方程，可通过消去 (x,y)(x,y)(x,y) 得到。

  • 如果仅是平移 w=z+bw=z + bw=z+b，那 (x,y)↦(x+γ, y+δ)(x,y)\mapsto (x+\gamma,\,y+\delta)(x,y)↦(x+γ,y+δ)，边界线方程非常容易转换：

    • x=cx=cx=c 变为 u=γ+cu=\gamma + cu=γ+c；
    • y=cy=cy=c 变为 v=δ+cv=\delta + cv=δ+c；
    • y=mx+ny=mx + ny=mx+n 变为 v=δ+m(u−γ)+nv = \delta + m(u-\gamma) + nv=δ+m(u−γ)+n 之类的简单直线方程。

示例（第一问）

w=z+(1−3i),w = z + (1 - \sqrt{3}i), w=z+(1−3​i),

则 a=1a=1a=1，b=1−3ib=1-\sqrt{3}ib=1−3​i。记 z=x+iyz=x+iyz=x+iy，w=u+ivw=u+ivw=u+iv。由平移可知

u=x+1,v=y−3.u = x + 1, \quad v = y - \sqrt{3}. u=x+1,v=y−3​.

边界直线：

  1. x=1  ⟹  u=2x = 1 \implies u = 2x=1⟹u=2;
  2. y=1  ⟹  v=1−3y = 1 \implies v = 1 - \sqrt{3}y=1⟹v=1−3​;
  3. y=1−x  ⟹  v+3=1−(u−1)y = 1 - x \implies v + \sqrt{3} = 1 - (u-1)y=1−x⟹v+3​=1−(u−1)
      ⟹  v+3=2−u  ⟹  u+v=2−3.\implies v + \sqrt{3} = 2 - u \implies u + v = 2 - \sqrt{3}.⟹v+3​=2−u⟹u+v=2−3​.
    这三条即是 www-平面的三条直线方程。

二、非线性变换

若

w=f(z)w = f(z) w=f(z)

不是单纯的线性/仿射（如 w=z2w=z^2w=z2、w=1zw = \frac{1}{z}w=z1​ 等），那直线一般会映射成圆或二次曲线。常用做法是：

  1. 给直线参数
    比如本题中边界线 x=1x=1x=1，可写成

    z=1+it,t∈R.z = 1 + i t,\quad t\in \mathbb{R}. z=1+it,t∈R.

    然后代入 w=f(z)w=f(z)w=f(z)。如果能将 ttt 消去，就能得到 www-平面的隐式方程；或者也可以直接把它当成参数方程。

  2. 找顶点映射
    由于是三角形，映射后依然可以在曲线上找到对应的顶点，并将三条曲线连接起来。

示例（第三问）

w=z2,z=x+iy.w=z^2, \quad z=x+iy. w=z2,z=x+iy.

  • 对边界 x=1x=1x=1，令 z=1+itz = 1 + i tz=1+it。

    w=(1+it)2=1+2it−t2=(1−t2)+i (2t).w = (1 + i t)^2 = 1 + 2 i t - t^2 = (1 - t^2) + i\,(2t). w=(1+it)2=1+2it−t2=(1−t2)+i(2t).

    记 w=u+ivw = u + i vw=u+iv，则

    u=1−t2,v=2t.u = 1 - t^2,\quad v = 2t. u=1−t2,v=2t.

    消去 ttt 可得 v=2t  ⟹  t=v2v = 2t \implies t=\tfrac{v}{2}v=2t⟹t=2v​，再带回 u=1−(v2)2u=1 - \bigl(\frac{v}{2}\bigr)^2u=1−(2v​)2，即

    u=1−v24,u = 1 - \frac{v^2}{4}, u=1−4v2​,

    在 www-平面是抛物线。同理可处理 y=1y=1y=1 和 y=1−xy=1-xy=1−x 两条边界。

小结

  1. 线性/仿射变换：

    • 直线 ↦\mapsto↦ 直线，圆 ↦\mapsto↦ 圆/直线，方便直接写出新方程；
    • 可先把顶点坐标带入 w=az+bw = a z + bw=az+b，也可以将直线方程 αx+βy+γ=0\alpha x + \beta y + \gamma=0αx+βy+γ=0 直接转换。

  2. 非线性变换：

    • 多用“参数法”或“消元法”获取映射曲线；
    • 注重顶点映射，结合三条边界映射曲线画出新的区域形状。

以上是一般化的基本思路。先理解“直线 ↦\mapsto↦ 直线/曲线”的规律，再针对线性与非线性分别处理，就能有条不紊地求得边界方程与映射后的区域。

Cyletix

我有一个疑问， 这里能这么做，其实是假设w能够写成w=u+ivw=u+ivw=u+iv的形式，但是你怎么知道一个复平面下变换后的新坐标下，曲线能写成这样？是否会出现实部和虚部不能分成加法的形式的可能性？

当我们说“w=u+i vw = u + i\,vw=u+iv”时，实际上利用的是“复数与R2\mathbb{R}^2R2的一一对应”。不管你怎么把zzz映射到www，只要www还在同样的复平面（或者说是C\mathbb{C}C本身，而不是跳到别的更抽象的空间），那就一定能找到“实部”和“虚部”两个实数，使得

w=u+i v,u,v∈R.w = u + i\,v,\quad u,v\in \mathbb{R}. w=u+iv,u,v∈R.

这是复数最基础的定义特性：任何复数都可以分解成“实数部分”加“虚数部分乘iii”的形式。

在你提到的映射或“坐标变换”场景中，出现的典型映射诸如

w=az+b,w=z2,w=ez,…w = a z + b, \quad w = z^2, \quad w = e^z, \quad \ldots w=az+b,w=z2,w=ez,…

它们最终的像w\displaystyle ww仍然是一个复数，因此依然可以写成

w=ℜ(w)⏟=u+i ℑ(w)⏟=v,w = \underbrace{\Re(w)}_{=u} + i\,\underbrace{\Im(w)}_{=v}, w==uℜ(w)​​+i=vℑ(w)​​,

不会出现“实部和虚部不能分成加法的形式”这种情况。

简而言之：

  1. 只要映射结果还在C\mathbb{C}C中，www就一定是个复数。
  2. 任何复数www都能无歧义地分解成u+i vu + i\,vu+iv。
  3. 在做几何或代数运算时，ℜ(w)\Re(w)ℜ(w)与ℑ(w)\Im(w)ℑ(w)可视为www在R2\mathbb{R}^2R2中的两个坐标分量，这和普通的“(x,y)(x,y)(x,y)在二维坐标系中”是同一个道理。

因此，你看到我们在做“w=z2w = z^2w=z2”之类的变换时，最后总能写出

u(t)=ℜ(z(t)2),v(t)=ℑ(z(t)2).u(t) = \Re\bigl(z(t)^2\bigr),\quad v(t) = \Im\bigl(z(t)^2\bigr). u(t)=ℜ(z(t)2),v(t)=ℑ(z(t)2).

并不需要额外假设“能拆”或“不能拆”；这是复数天生就带有的一种结构。只要还是在复平面里，www的实部和虚部就一定存在且能分离出来。
