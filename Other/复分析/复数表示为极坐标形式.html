复数表示为极坐标形式

2025/10/20数学

要将任意复数 z=x+iyz = x + iyz=x+iy 转换为极坐标形式 z=r(cos⁡θ+isin⁡θ)z = r(\cos \theta + i\sin \theta)z=r(cosθ+isinθ) 或 z=reiθz = re^{i\theta}z=reiθ, 需要进行以下步骤:

1. 计算模 rrr

复数的模 rrr 表示复数在复平面上与原点之间的距离, 计算公式为:

r=∣z∣=x2+y2 r = |z| = \sqrt{x^2 + y^2} r=∣z∣=x2+y2​

其中 xxx 是实部, yyy 是虚部.

2. 计算辐角 θ\thetaθ

辐角 θ\thetaθ 是复数相对于实轴的角度, 通常用弧度表示, 计算公式为:

θ=tan⁡−1(yx) \theta = \tan^{-1}\left(\frac{y}{x}\right) θ=tan−1(xy​)

然而, 为了准确地确定 θ\thetaθ 的值, 特别是在复数位于不同象限时, 使用反正切函数时需要特别小心. 具体来说:

  • 当 x>0x > 0x>0 且 y≥0y \geq 0y≥0 时, θ=tan⁡−1(yx)\theta = \tan^{-1}\left(\frac{y}{x}\right)θ=tan−1(xy​)
  • 当 x<0x < 0x<0 时, θ=tan⁡−1(yx)+π\theta = \tan^{-1}\left(\frac{y}{x}\right) + \piθ=tan−1(xy​)+π
  • 当 x>0x > 0x>0 且 y<0y < 0y<0 时, θ=tan⁡−1(yx)\theta = \tan^{-1}\left(\frac{y}{x}\right)θ=tan−1(xy​) (结果为负角度)
  • 当 x=0x = 0x=0 且 y>0y > 0y>0 时, θ=π2\theta = \frac{\pi}{2}θ=2π​
  • 当 x=0x = 0x=0 且 y<0y < 0y<0 时, θ=−π2\theta = -\frac{\pi}{2}θ=−2π​
    注意: 在很多编程语言中, 会有专门的函数 atan2(y, x), 它可以直接处理这些情况并返回正确的辐角.

3. 写出极坐标形式

现在, 你可以将复数 z=x+iyz = x + iyz=x+iy 表示为极坐标形式 z=r(cos⁡θ+isin⁡θ)z = r(\cos \theta + i\sin \theta)z=r(cosθ+isinθ) 或者 z=reiθz = re^{i\theta}z=reiθ, 其中:

z=r(cos⁡θ+isin⁡θ)=reiθ z = r\left(\cos \theta + i\sin \theta\right) = re^{i\theta} z=r(cosθ+isinθ)=reiθ

示例

例如, 假设 z=1+i3z = 1 + i\sqrt{3}z=1+i3​:

  1. 计算模:

    r=12+(3)2=1+3=4=2r = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1 + 3} = \sqrt{4} = 2 r=12+(3​)2​=1+3​=4​=2

  2. 计算辐角:

    θ=tan⁡−1(31)=tan⁡−1(3)=π3\theta = \tan^{-1}\left(\frac{\sqrt{3}}{1}\right) = \tan^{-1}(\sqrt{3}) = \frac{\pi}{3} θ=tan−1(13​​)=tan−1(3​)=3π​

  3. 极坐标形式:

    z=2(cos⁡π3+isin⁡π3)=2eiπ3z = 2\left(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3}\right) = 2e^{i\frac{\pi}{3}} z=2(cos3π​+isin3π​)=2ei3π​

通过这些步骤, 你可以将任意复数转换为其极坐标形式.
