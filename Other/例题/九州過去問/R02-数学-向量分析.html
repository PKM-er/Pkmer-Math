R02-数学-向量分析

2025/10/18

3. ベクトル解析 Vector analysis

在直角坐标系中，沿 x,y,zx, y, zx,y,z 轴方向的单位矢量分别为 i,j,ki, j, ki,j,k，回答以下问题：

  1. 考虑面 z=x2+y2z = x^2 + y^2z=x2+y2 和 面 z=(x−12)2+(y−12)2z = \left(x - \frac{1}{\sqrt{2}}\right)^2 + \left(y - \frac{1}{\sqrt{2}}\right)^2z=(x−2​1​)2+(y−2​1​)2：
    a. 证明点 P(24,24,14)P\left(\frac{\sqrt{2}}{4}, \frac{\sqrt{2}}{4}, \frac{1}{4}\right)P(42​​,42​​,41​) 均在两面上。
    b. 求点 PPP 处两面的法线的夹角。
  2. 定义矢量场 A=x2i−y2j+z2kA = x^2i - y^2j + z^2kA=x2i−y2j+z2k，令 SSS 为由 x2+y2=9,z=0,z=4x^2 + y^2 = 9, z = 0, z = 4x2+y2=9,z=0,z=4 包围的圆柱的表面，计算面积分

    ∬SA⋅n dS,\iint_S A \cdot n \, dS, ∬S​A⋅ndS,

    其中 nnn 为 SSS 的外向单位法向量。

解答

1. 两曲面的法向量

a. 证明点 PPP 均在两面上

点 P(24,24,14)P\left(\frac{\sqrt{2}}{4}, \frac{\sqrt{2}}{4}, \frac{1}{4}\right)P(42​​,42​​,41​) 是否满足两个面的方程：

  1. 面 z=x2+y2z = x^2 + y^2z=x2+y2：
    将 x=24,y=24,z=14x = \frac{\sqrt{2}}{4}, y = \frac{\sqrt{2}}{4}, z = \frac{1}{4}x=42​​,y=42​​,z=41​ 代入：

    z=(24)2+(24)2=18+18=14.z = \left(\frac{\sqrt{2}}{4}\right)^2 + \left(\frac{\sqrt{2}}{4}\right)^2 = \frac{1}{8} + \frac{1}{8} = \frac{1}{4}. z=(42​​)2+(42​​)2=81​+81​=41​.

    符合该面方程。
  2. 面 z=(x−12)2+(y−12)2z = \left(x - \frac{1}{\sqrt{2}}\right)^2 + \left(y - \frac{1}{\sqrt{2}}\right)^2z=(x−2​1​)2+(y−2​1​)2：
    将 x=24,y=24,z=14x = \frac{\sqrt{2}}{4}, y = \frac{\sqrt{2}}{4}, z = \frac{1}{4}x=42​​,y=42​​,z=41​ 代入：

    z=(24−12)2+(24−12)2.z = \left(\frac{\sqrt{2}}{4} - \frac{1}{\sqrt{2}}\right)^2 + \left(\frac{\sqrt{2}}{4} - \frac{1}{\sqrt{2}}\right)^2. z=(42​​−2​1​)2+(42​​−2​1​)2.

    计算 24−12=−24\frac{\sqrt{2}}{4} - \frac{1}{\sqrt{2}} = -\frac{\sqrt{2}}{4}42​​−2​1​=−42​​：

    z=(−24)2+(−24)2=18+18=14.z = \left(-\frac{\sqrt{2}}{4}\right)^2 + \left(-\frac{\sqrt{2}}{4}\right)^2 = \frac{1}{8} + \frac{1}{8} = \frac{1}{4}. z=(−42​​)2+(−42​​)2=81​+81​=41​.

    符合该面方程。
    因此，点 PPP 在两面上。

b. 求点 PPP 处两面的法线夹角

法线向量是梯度 ∇f\nabla f∇f 的方向，分别计算两面的法线向量：

  1. 面 z=x2+y2z = x^2 + y^2z=x2+y2 对应隐函数 f1(x,y,z)=z−x2−y2f_1(x, y, z) = z - x^2 - y^2f1​(x,y,z)=z−x2−y2：

    ∇f1=(−2x,−2y,1).\nabla f_1 = \left(-2x, -2y, 1\right). ∇f1​=(−2x,−2y,1).

  2. 面 z=(x−12)2+(y−12)2z = \left(x - \frac{1}{\sqrt{2}}\right)^2 + \left(y - \frac{1}{\sqrt{2}}\right)^2z=(x−2​1​)2+(y−2​1​)2 对应隐函数 f2(x,y,z)=z−(x−12)2−(y−12)2f_2(x, y, z) = z - \left(x - \frac{1}{\sqrt{2}}\right)^2 - \left(y - \frac{1}{\sqrt{2}}\right)^2f2​(x,y,z)=z−(x−2​1​)2−(y−2​1​)2：

    ∇f2=(−2(x−12),−2(y−12),1).\nabla f_2 = \left(-2\left(x - \frac{1}{\sqrt{2}}\right), -2\left(y - \frac{1}{\sqrt{2}}\right), 1\right). ∇f2​=(−2(x−2​1​),−2(y−2​1​),1).

在点 PPP 处 (x=24,y=24,z=14)\left(x = \frac{\sqrt{2}}{4}, y = \frac{\sqrt{2}}{4}, z = \frac{1}{4}\right)(x=42​​,y=42​​,z=41​)，计算法线向量：

  • ∇f1\nabla f_1∇f1​：

    ∇f1=(−2⋅24,−2⋅24,1)=(−22,−22,1).\nabla f_1 = \left(-2 \cdot \frac{\sqrt{2}}{4}, -2 \cdot \frac{\sqrt{2}}{4}, 1\right) = \left(-\frac{\sqrt{2}}{2}, -\frac{\sqrt{2}}{2}, 1\right). ∇f1​=(−2⋅42​​,−2⋅42​​,1)=(−22​​,−22​​,1).

  • ∇f2\nabla f_2∇f2​：

    ∇f2=(−2(24−12),−2(24−12),1).\nabla f_2 = \left(-2\left(\frac{\sqrt{2}}{4} - \frac{1}{\sqrt{2}}\right), -2\left(\frac{\sqrt{2}}{4} - \frac{1}{\sqrt{2}}\right), 1\right). ∇f2​=(−2(42​​−2​1​),−2(42​​−2​1​),1).

    计算 24−12=−24\frac{\sqrt{2}}{4} - \frac{1}{\sqrt{2}} = -\frac{\sqrt{2}}{4}42​​−2​1​=−42​​：

    ∇f2=(−2⋅−24,−2⋅−24,1)=(22,22,1).\nabla f_2 = \left(-2 \cdot -\frac{\sqrt{2}}{4}, -2 \cdot -\frac{\sqrt{2}}{4}, 1\right) = \left(\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2}, 1\right). ∇f2​=(−2⋅−42​​,−2⋅−42​​,1)=(22​​,22​​,1).

两向量的夹角由公式 cos⁡θ=∇f1⋅∇f2∥∇f1∥∥∇f2∥\cos \theta = \frac{\nabla f_1 \cdot \nabla f_2}{\|\nabla f_1\| \|\nabla f_2\|}cosθ=∥∇f1​∥∥∇f2​∥∇f1​⋅∇f2​​ 给出：

  1. 向量点积：

    ∇f1⋅∇f2=(−22)(22)+(−22)(22)+1⋅1=−12−12+1=0.\nabla f_1 \cdot \nabla f_2 = \left(-\frac{\sqrt{2}}{2}\right)\left(\frac{\sqrt{2}}{2}\right) + \left(-\frac{\sqrt{2}}{2}\right)\left(\frac{\sqrt{2}}{2}\right) + 1 \cdot 1 = -\frac{1}{2} - \frac{1}{2} + 1 = 0. ∇f1​⋅∇f2​=(−22​​)(22​​)+(−22​​)(22​​)+1⋅1=−21​−21​+1=0.

  2. 向量模长：

    ∥∇f1∥=(−22)2+(−22)2+12=12+12+1=2.\|\nabla f_1\| = \sqrt{\left(-\frac{\sqrt{2}}{2}\right)^2 + \left(-\frac{\sqrt{2}}{2}\right)^2 + 1^2} = \sqrt{\frac{1}{2} + \frac{1}{2} + 1} = \sqrt{2}. ∥∇f1​∥=(−22​​)2+(−22​​)2+12​=21​+21​+1​=2​.

    ∥∇f2∥=(22)2+(22)2+12=12+12+1=2.\|\nabla f_2\| = \sqrt{\left(\frac{\sqrt{2}}{2}\right)^2 + \left(\frac{\sqrt{2}}{2}\right)^2 + 1^2} = \sqrt{\frac{1}{2} + \frac{1}{2} + 1} = \sqrt{2}. ∥∇f2​∥=(22​​)2+(22​​)2+12​=21​+21​+1​=2​.

夹角：

cos⁡θ=02⋅2=0⇒θ=π2.\cos \theta = \frac{0}{\sqrt{2} \cdot \sqrt{2}} = 0 \quad \Rightarrow \quad \theta = \frac{\pi}{2}. cosθ=2​⋅2​0​=0⇒θ=2π​.

结论：两面法线夹角为 π2\frac{\pi}{2}2π​，即两面正交。

2. 圆柱面积分

圆柱表面 SSS

圆柱 x2+y2=9,z=0,z=4x^2 + y^2 = 9, z = 0, z = 4x2+y2=9,z=0,z=4 包括底面、顶面和侧面，分别计算：

1. 底面 z=0z = 0z=0

法向量 n=−kn = -kn=−k，dS=dx dydS = dx \, dydS=dxdy。在底面 A=x2i−y2jA = x^2i - y^2jA=x2i−y2j，计算通量：

A⋅n=(x2i−y2j+z2k)⋅(−k)=0.A \cdot n = (x^2i - y^2j + z^2k) \cdot (-k) = 0. A⋅n=(x2i−y2j+z2k)⋅(−k)=0.

底面贡献为 000。

2. 顶面 z=4z = 4z=4

法向量 n=kn = kn=k，dS=dx dydS = dx \, dydS=dxdy。在顶面 A=x2i−y2j+16kA = x^2i - y^2j + 16kA=x2i−y2j+16k，计算通量：

A⋅n=(x2i−y2j+z2k)⋅k=16.A \cdot n = (x^2i - y^2j + z^2k) \cdot k = 16. A⋅n=(x2i−y2j+z2k)⋅k=16.

顶面面积为 πr2=π⋅9=9π\pi r^2 = \pi \cdot 9 = 9\piπr2=π⋅9=9π，积分结果：

∬顶面A⋅n dS=16⋅9π=144π.\iint_{\text{顶面}} A \cdot n \, dS = 16 \cdot 9\pi = 144\pi. ∬顶面​A⋅ndS=16⋅9π=144π.

3. 侧面 x2+y2=9x^2 + y^2 = 9x2+y2=9

侧面法向量 n=(x,y,0)/x2+y2=(x,y,0)/3n = (x, y, 0)/\sqrt{x^2 + y^2} = (x, y, 0)/3n=(x,y,0)/x2+y2​=(x,y,0)/3，dS=3 dθ dzdS = 3 \, d\theta \, dzdS=3dθdz。在侧面 A=x2i−y2j+z2kA = x^2i - y^2j + z^2kA=x2i−y2j+z2k，计算通量：

A⋅n=(x2,−y2,z2)⋅(x,y,0)3=x3−y33.A \cdot n = (x^2, -y^2, z^2) \cdot \frac{(x, y, 0)}{3} = \frac{x^3 - y^3}{3}. A⋅n=(x2,−y2,z2)⋅3(x,y,0)​=3x3−y3​.

积分对称，x3x^3x3 和 y3y^3y3 分量关于 θ\thetaθ 的积分为 000，侧面贡献为 000。

总结

∬SA⋅n dS=0+144π+0=144π.\iint_S A \cdot n \, dS = 0 + 144\pi + 0 = 144\pi. ∬S​A⋅ndS=0+144π+0=144π.

高斯公式

也可以使用高斯公式，即散度定理：

∬SA⋅n dS=∭V(∇⋅A) dV\iint_S A \cdot n \, dS = \iiint_V (\nabla \cdot A) \, dV ∬S​A⋅ndS=∭V​(∇⋅A)dV

其中 VVV 是该圆柱的体积。

  1. 散度计算：
    计算矢量场 AAA 的散度：

∇⋅A=∂Ax∂x+∂Ay∂y+∂Az∂z\nabla \cdot A = \frac{\partial A_x}{\partial x} + \frac{\partial A_y}{\partial y} + \frac{\partial A_z}{\partial z} ∇⋅A=∂x∂Ax​​+∂y∂Ay​​+∂z∂Az​​

Ax=x2,Ay=−y2,Az=z2A_x = x^2, \quad A_y = -y^2, \quad A_z = z^2 Ax​=x2,Ay​=−y2,Az​=z2

∇⋅A=∂(x2)∂x+∂(−y2)∂y+∂(z2)∂z=2x+(−2y)+2z=2(x−y+z)\nabla \cdot A = \frac{\partial (x^2)}{\partial x} + \frac{\partial (-y^2)}{\partial y} + \frac{\partial (z^2)}{\partial z} = 2x + (-2y) + 2z = 2(x - y + z) ∇⋅A=∂x∂(x2)​+∂y∂(−y2)​+∂z∂(z2)​=2x+(−2y)+2z=2(x−y+z)

  1. 积分区域：
    圆柱由方程 x2+y2=9x^2 + y^2 = 9x2+y2=9 定义，表示半径为 3 的圆柱，zzz 的范围是从 000 到 444。

  2. 体积分计算：
    将积分转化为柱坐标系计算：

x=rcos⁡θ,y=rsin⁡θ,z=zx = r \cos \theta, \quad y = r \sin \theta, \quad z = z x=rcosθ,y=rsinθ,z=z

散度在柱坐标下表达为：

∇⋅A=2(rcos⁡θ−rsin⁡θ+z)\nabla \cdot A = 2(r \cos \theta - r \sin \theta + z) ∇⋅A=2(rcosθ−rsinθ+z)

体积元素为：

dV=r dr dθ dzdV = r \, dr \, d\theta \, dz dV=rdrdθdz

积分范围为：

0≤r≤3,0≤θ≤2π,0≤z≤40 \leq r \leq 3, \quad 0 \leq \theta \leq 2\pi, \quad 0 \leq z \leq 4 0≤r≤3,0≤θ≤2π,0≤z≤4

进行积分：

∭V2(rcos⁡θ−rsin⁡θ+z) r dr dθ dz\iiint_V 2(r \cos \theta - r \sin \theta + z) \, r \, dr \, d\theta \, dz ∭V​2(rcosθ−rsinθ+z)rdrdθdz

计算过程：

∭V2(rcos⁡θ−rsin⁡θ+z)r dr dθ dz\iiint_V 2(r \cos \theta - r \sin \theta + z) r \, dr \, d\theta \, dz ∭V​2(rcosθ−rsinθ+z)rdrdθdz

分解积分：

2∫04dz∫02πdθ∫03(r2cos⁡θ−r2sin⁡θ+zr)dr2 \int_0^4 dz \int_0^{2\pi} d\theta \int_0^3 (r^2 \cos \theta - r^2 \sin \theta + zr) dr 2∫04​dz∫02π​dθ∫03​(r2cosθ−r2sinθ+zr)dr

分步计算：

∫03r2dr=[r33]03=9\int_0^3 r^2 dr = \left[ \frac{r^3}{3} \right]_0^3 = 9 ∫03​r2dr=[3r3​]03​=9

∫02πcos⁡θdθ=0,∫02πsin⁡θdθ=0,∫02πdθ=2π\int_0^{2\pi} \cos \theta d\theta = 0, \quad \int_0^{2\pi} \sin \theta d\theta = 0, \quad \int_0^{2\pi} d\theta = 2\pi ∫02π​cosθdθ=0,∫02π​sinθdθ=0,∫02π​dθ=2π

∫03rdr=[r22]03=92\int_0^3 r dr = \left[ \frac{r^2}{2} \right]_0^3 = \frac{9}{2} ∫03​rdr=[2r2​]03​=29​

∫04zdz=[z22]04=8\int_0^4 z dz = \left[ \frac{z^2}{2} \right]_0^4 = 8 ∫04​zdz=[2z2​]04​=8

组合后：
