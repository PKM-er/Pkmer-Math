4. 绘制椭圆

利用这些参数，我们可以绘制椭圆：

import numpy as np
import matplotlib.pyplot as plt

# 矩阵 P
P = np.array([[7, np.sqrt(3)],
              [np.sqrt(3), 5]])

# 特征值和特征向量
eigvals, eigvecs = np.linalg.eigh(P)

# 半长轴和半短轴
a = 1 / np.sqrt(eigvals[1])
b = 1 / np.sqrt(eigvals[0])

# 生成椭圆点
theta = np.linspace(0, 2 * np.pi, 100)
ellipse = np.array([a * np.cos(theta), b * np.sin(theta)])

# 旋转椭圆
R = eigvecs
rotated_ellipse = R @ ellipse

# 绘制椭圆
plt.plot(rotated_ellipse[0, :], rotated_ellipse[1, :], label='Ellipse')
plt.xlabel('x')
plt.ylabel('y')
plt.axis('equal')
plt.grid()
plt.legend()
plt.title('$x^T P x = 8$')
plt.show()

通过这些步骤，可以从二次型 ϕ(x)=xTPx=8\phi(x) = x^T P x = 8ϕ(x)=xTPx=8 中提取出椭圆的特征并绘制出相应的图形。

二次型 xTPxx^T P xxTPx 可以表示一个椭圆，但这取决于矩阵 PPP 的性质以及二次型方程右边的常数。在二维情况下，如果 PPP 是正定的，二次型 xTPx=cx^T P x = cxTPx=c（其中 ccc 是一个正数）描述的是一个椭圆。

椭圆的方程

在二维空间中，如果 PPP 是正定矩阵，那么 xTPx=cx^T P x = cxTPx=c 描述的是一个椭圆。假设 PPP 经过特征值分解可以表示为 P=QΛQTP = Q \Lambda Q^TP=QΛQT，其中 QQQ 是包含 PPP 的特征向量的正交矩阵，Λ\LambdaΛ 是对角线上为 PPP 的特征值的对角矩阵。此时，二次型可以重写为：

xTPx=xTQΛQTx=(QTx)TΛ(QTx) x^T P x = x^T Q \Lambda Q^T x = (Q^T x)^T \Lambda (Q^T x) xTPx=xTQΛQTx=(QTx)TΛ(QTx)

设 y=QTxy = Q^T xy=QTx，则有：

yTΛy=λ1y12+λ2y22=c y^T \Lambda y = \lambda_1 y_1^2 + \lambda_2 y_2^2 = c yTΛy=λ1​y12​+λ2​y22​=c

其中 λ1,λ2\lambda_1, \lambda_2λ1​,λ2​ 是 Λ\LambdaΛ 对角线上的元素（PPP 的特征值），而 y1,y2y_1, y_2y1​,y2​ 是变换后的坐标系中的坐标。将上述方程整理可得：

y12c/λ1+y22c/λ2=1 \frac{y_1^2}{c/\lambda_1} + \frac{y_2^2}{c/\lambda_2} = 1 c/λ1​y12​​+c/λ2​y22​​=1

这是一个标准的椭圆方程，其中半轴的长度分别为 c/λ1\sqrt{c/\lambda_1}c/λ1​​ 和 c/λ2\sqrt{c/\lambda_2}c/λ2​​。

椭圆方程的右边

在椭圆的标准方程中，右边通常是 1。如果右边是其他数字 ddd，则方程变为：

x2a2+y2b2=d \frac{x^2}{a^2} + \frac{y^2}{b^2} = d a2x2​+b2y2​=d

为了将其标准化成椭圆方程，我们可以除以 ddd：

x2a2/d+y2b2/d=1 \frac{x^2}{a^2/d} + \frac{y^2}{b^2/d} = 1 a2/dx2​+b2/dy2​=1

这意味着实际的半轴变为 a2/d\sqrt{a^2/d}a2/d​ 和 b2/d\sqrt{b^2/d}b2/d​。当 ddd 增大时，椭圆变小；当 ddd 减小时，椭圆变大。

椭圆的变化

如果方程右边的常数 ccc 变化，则椭圆的大小相应变化。更具体地说：

  • 当 ccc 增加时，椭圆的大小增大，因为半轴 c/λ1\sqrt{c/\lambda_1}c/λ1​​ 和 c/λ2\sqrt{c/\lambda_2}c/λ2​​ 都会变大。
  • 当 ccc 减少时，椭圆的大小减小。

这种方式通过改变等式右边的常数 ccc，直接影响了椭圆的尺寸，但保持了其形状和方向。

当椭圆方程的右边从标准的 1 变为其他数值（例如 8），这实际上对椭圆的大小有直接的影响。这里，我们考虑的方程是：

x2a2/d+y2b2/d=8 \frac{x^2}{a^2/d} + \frac{y^2}{b^2/d} = 8 a2/dx2​+b2/dy2​=8

调整方程的形式

首先，将方程整理为标准的椭圆形式，我们可以通过将方程两边除以 8 来实现：

x2(a2/d)/8+y2(b2/d)/8=1 \frac{x^2}{(a^2/d)/8} + \frac{y^2}{(b^2/d)/8} = 1 (a2/d)/8x2​+(b2/d)/8y2​=1

简化一下得：

x2a2/(8d)+y2b2/(8d)=1 \frac{x^2}{a^2/(8d)} + \frac{y^2}{b^2/(8d)} = 1 a2/(8d)x2​+b2/(8d)y2​=1

椭圆的变化解释

上面的方程表明椭圆的新半轴长度为：

  • 半长轴的长度：a2/(8d)\sqrt{a^2/(8d)}a2/(8d)​
  • 半短轴的长度：b2/(8d)\sqrt{b^2/(8d)}b2/(8d)​

对椭圆的实际影响

当方程右边的数值由 1 变为 8 时，根据上述调整后的方程，椭圆的半长轴和半短轴实际上都变小了。具体来说：

  • 半长轴和半短轴都被除以 8\sqrt{8}8​，这意味着椭圆的每个轴长都缩小了。
  • 这种缩小等同于将每个轴长乘以 18≈0.354\frac{1}{\sqrt{8}} \approx 0.3548​1​≈0.354，相比原来的尺寸缩小了。

几何直观

从几何上看，如果原来的椭圆方程是 x2a2+y2b2=1\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1a2x2​+b2y2​=1，这个椭圆拥有半轴 aaa 和 bbb。将方程右边改为 8 后，实际上是将整个椭圆沿着原点对称地缩小了 8\sqrt{8}8​ 倍，得到了一个更小的椭圆，其半轴为 a8\frac{a}{\sqrt{8}}8​a​ 和 b8\frac{b}{\sqrt{8}}8​b​。

这种变换可以在各种图形处理软件或者通过数学工具进行可视化，以更直观地理解椭圆的这种变化。

从特征值到椭圆的半轴长度

对于二次型方程 xTPx=cx^T P x = cxTPx=c，其与椭圆方程的对应关系为：

y12c/λ1+y22c/λ2=1 \frac{y_1^2}{c/\lambda_1} + \frac{y_2^2}{c/\lambda_2} = 1 c/λ1​y12​​+c/λ2​y22​​=1

其中，y1y_1y1​ 和 y2y_2y2​ 是在特征向量定义的新坐标系中的坐标。对于这种情况：

  • 半长轴 aaa 对应较小的特征值（因为较小的特征值对应较大的分母，从而产生较长的轴），即 λ2=4\lambda_2 = 4λ2​=4。
  • 半短轴 bbb 对应较大的特征值（因为较大的特征值对应较小的分母，从而产生较短的轴），即 λ1=8\lambda_1 = 8λ1​=8。

因此：

a=84=2,b=88=1 a = \sqrt{\frac{8}{4}} = \sqrt{2}, \quad b = \sqrt{\frac{8}{8}} = 1 a=48​​=2​,b=88​​=1

解释

  • 特征值 λ1=8\lambda_1 = 8λ1​=8 和 λ2=4\lambda_2 = 4λ2​=4 表示在各自的特征向量方向上，二次型方程形成的椭圆被拉伸的程度。具体来说，λ2=4\lambda_2 = 4λ2​=4 对应的方向上拉伸较少（因为 c/λ2=2c/\lambda_2 = 2c/λ2​=2），而 λ1=8\lambda_1 = 8λ1​=8 对应的方向上拉伸更多（因为 c/λ1=1c/\lambda_1 = 1c/λ1​=1）。
  • λ1=8\lambda_1 = 8λ1​=8 表示在这个方向上的拉伸因子是 8/c=1=1\sqrt{8/c} = \sqrt{1} = 18/c​=1​=1，这意味着在此方向上长度没有放缩，保持原始大小。
  • λ2=4\lambda_2 = 4λ2​=4 表示在这个方向上的拉伸因子是 8/c=2\sqrt{8/c} = \sqrt{2}8/c​=2​，这意味着在此方向上长度放大了 2\sqrt{2}2​ 倍。

几何直观

这种几何直观的解释是基于标准椭圆的尺寸和特征值的倒数关系。较大的特征值导致较小的分母，因此在这个方向上的拉伸程度较小，产生较短的轴；而较小的特征值则导致较大的分母，因此在这个方向上的拉伸程度较大，产生较长的轴。这解释了二次型的特征值如何影响椭圆形状的拉伸。
