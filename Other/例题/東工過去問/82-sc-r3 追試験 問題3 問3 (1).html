дёәдәҶжүҫеҲ°жңҖе°Өдј°и®Ў(MLE) , жҲ‘д»¬йңҖиҰҒеҜ№ ОІ1\beta_1ОІ1вҖӢ е’Ң ОІ2\beta_2ОІ2вҖӢ жұӮеҜј, е№¶дҪҝеҫ—еҜјж•°дёәйӣ¶. з”ұдәҺ LLL жҳҜдёҖдёӘд№ҳз§Ҝзҡ„жҢҮж•°еҮҪж•°, еӨ„зҗҶиө·жқҘдёҚж–№дҫҝ, жҲ‘д»¬йҖҡеёёеҸ–еҜ№ж•°е°ӨеәҰеҮҪж•°(log-likelihood function) :

logвҒЎL(ОІ1,ОІ2)=вҲ’32logвҒЎ(2ПҖ)вҲ’12[(вҲ’5вҲ’ОІ1)2+(7вҲ’ОІ2)2+(1вҲ’(ОІ1+ОІ2))2]\log L(\beta_1, \beta_2) = -\frac{3}{2} \log(2\pi) - \frac{1}{2} \left[(-5 - \beta_1)^2 + (7 - \beta_2)^2 + (1 - (\beta_1 + \beta_2))^2\right] logL(ОІ1вҖӢ,ОІ2вҖӢ)=вҲ’23вҖӢlog(2ПҖ)вҲ’21вҖӢ[(вҲ’5вҲ’ОІ1вҖӢ)2+(7вҲ’ОІ2вҖӢ)2+(1вҲ’(ОІ1вҖӢ+ОІ2вҖӢ))2]

еҜ№ж•°е°ӨеәҰеҮҪж•°зҡ„жңҖеӨ§еҢ–зӯүеҗҢдәҺе°ӨеәҰеҮҪж•°зҡ„жңҖеӨ§еҢ–. жҲ‘д»¬еҲҶеҲ«еҜ№ ОІ1\beta_1ОІ1вҖӢ е’Ң ОІ2\beta_2ОІ2вҖӢ жұӮеҒҸеҜјж•°, е№¶и®ҫдёәйӣ¶:

еҜ№ ОІ1\beta_1ОІ1вҖӢ жұӮеҒҸеҜјж•°:

вҲӮlogвҒЎLвҲӮОІ1=вҲ’вҲӮвҲӮОІ1[12((вҲ’5вҲ’ОІ1)2+(1вҲ’(ОІ1+ОІ2))2)]=0\frac{\partial \log L}{\partial \beta_1} = -\frac{\partial}{\partial \beta_1} \left[ \frac{1}{2} ((-5 - \beta_1)^2 + (1 - (\beta_1 + \beta_2))^2) \right] = 0 вҲӮОІ1вҖӢвҲӮlogLвҖӢ=вҲ’вҲӮОІ1вҖӢвҲӮвҖӢ[21вҖӢ((вҲ’5вҲ’ОІ1вҖӢ)2+(1вҲ’(ОІ1вҖӢ+ОІ2вҖӢ))2)]=0

еҜ№ ОІ2\beta_2ОІ2вҖӢ жұӮеҒҸеҜјж•°:

вҲӮlogвҒЎLвҲӮОІ2=вҲ’вҲӮвҲӮОІ2[12((7вҲ’ОІ2)2+(1вҲ’(ОІ1+ОІ2))2)]=0\frac{\partial \log L}{\partial \beta_2} = -\frac{\partial}{\partial \beta_2} \left[ \frac{1}{2} ((7 - \beta_2)^2 + (1 - (\beta_1 + \beta_2))^2) \right] = 0 вҲӮОІ2вҖӢвҲӮlogLвҖӢ=вҲ’вҲӮОІ2вҖӢвҲӮвҖӢ[21вҖӢ((7вҲ’ОІ2вҖӢ)2+(1вҲ’(ОІ1вҖӢ+ОІ2вҖӢ))2)]=0

и§ЈеҶіиҝҷдёӨдёӘж–№зЁӢеҸҜд»Ҙеҫ—еҲ° ОІ1\beta_1ОІ1вҖӢ е’Ң ОІ2\beta_2ОІ2вҖӢ зҡ„жңҖе°Өдј°и®Ў.

йҰ–е…ҲеҜ№ ОІ1\beta_1ОІ1вҖӢ зҡ„еҒҸеҜјж•°:

вҲӮlogвҒЎLвҲӮОІ1=вҲ’(вҲ’5вҲ’ОІ1)вҲ’(1вҲ’(ОІ1+ОІ2))=0\frac{\partial \log L}{\partial \beta_1} = -(-5 - \beta_1) - (1 - (\beta_1 + \beta_2)) = 0 вҲӮОІ1вҖӢвҲӮlogLвҖӢ=вҲ’(вҲ’5вҲ’ОІ1вҖӢ)вҲ’(1вҲ’(ОІ1вҖӢ+ОІ2вҖӢ))=0

вҮ’вҲ’5вҲ’ОІ1+1вҲ’ОІ1вҲ’ОІ2=0\Rightarrow -5 - \beta_1 + 1 - \beta_1 - \beta_2 = 0 вҮ’вҲ’5вҲ’ОІ1вҖӢ+1вҲ’ОІ1вҖӢвҲ’ОІ2вҖӢ=0

вҮ’вҲ’2ОІ1вҲ’ОІ2вҲ’4=0\Rightarrow -2\beta_1 - \beta_2 - 4 = 0 вҮ’вҲ’2ОІ1вҖӢвҲ’ОІ2вҖӢвҲ’4=0

вҮ’2ОІ1+ОІ2=вҲ’4(1)\Rightarrow 2\beta_1 + \beta_2 = -4 \quad \text{(1)} вҮ’2ОІ1вҖӢ+ОІ2вҖӢ=вҲ’4(1)

з„¶еҗҺеҜ№ ОІ2\beta_2ОІ2вҖӢ зҡ„еҒҸеҜјж•°:

вҲӮlogвҒЎLвҲӮОІ2=вҲ’(7вҲ’ОІ2)вҲ’(1вҲ’(ОІ1+ОІ2))=0\frac{\partial \log L}{\partial \beta_2} = -(7 - \beta_2) - (1 - (\beta_1 + \beta_2)) = 0 вҲӮОІ2вҖӢвҲӮlogLвҖӢ=вҲ’(7вҲ’ОІ2вҖӢ)вҲ’(1вҲ’(ОІ1вҖӢ+ОІ2вҖӢ))=0

вҮ’7вҲ’ОІ2+1вҲ’ОІ1вҲ’ОІ2=0\Rightarrow 7 - \beta_2 + 1 - \beta_1 - \beta_2 = 0 вҮ’7вҲ’ОІ2вҖӢ+1вҲ’ОІ1вҖӢвҲ’ОІ2вҖӢ=0

вҮ’вҲ’ОІ1вҲ’2ОІ2+8=0\Rightarrow -\beta_1 - 2\beta_2 + 8 = 0 вҮ’вҲ’ОІ1вҖӢвҲ’2ОІ2вҖӢ+8=0

вҮ’ОІ1+2ОІ2=8(2)\Rightarrow \beta_1 + 2\beta_2 = 8 \quad \text{(2)} вҮ’ОІ1вҖӢ+2ОІ2вҖӢ=8(2)

иҒ”з«Ӣж–№зЁӢ(1) е’Ң(2) еҸҜд»Ҙи§ЈеҮә ОІ1\beta_1ОІ1вҖӢ е’Ң ОІ2\beta_2ОІ2вҖӢ:

2ОІ1+ОІ2=вҲ’4ОІ1+2ОІ2=82\beta_1 + \beta_2 = -4 \\ \beta_1 + 2\beta_2 = 8 2ОІ1вҖӢ+ОІ2вҖӢ=вҲ’4ОІ1вҖӢ+2ОІ2вҖӢ=8

йҖҡиҝҮи§ЈиҝҷдёӘзәҝжҖ§ж–№зЁӢз»„, жҲ‘д»¬еҫ—еҲ°:

ОІ1=вҲ’6ОІ2=7\beta_1 = -6 \\ \beta_2 = 7 ОІ1вҖӢ=вҲ’6ОІ2вҖӢ=7

еӣ жӯӨ, жңҖе°Өдј°и®Ў ОІ1=вҲ’6\beta_1 = -6ОІ1вҖӢ=вҲ’6 е’Ң ОІ2=7\beta_2 = 7ОІ2вҖӢ=7.
