二阶常系数非齐次线性微分方程的一般形式

y′′+py′+qy=f(x)y''+py'+qy=f(x) y′′+py′+qy=f(x)

由二阶线性微分方程解的结构可知, 求解二阶常系数非齐次线性微分方程的通解, 归结为求对应齐次方程的通解和非齐次方程本身的一个特解

对应齐次方程的通解在常系数齐次线性微分方程得到解决, 所以这里只讨论求特解的方法.

复系数非齐次线性微分方程

待定系数法可以推广至复系数。推广时，复数形式的特解会包含实部和虚部。因此，方法的本质不会改变，只是处理的系数和解的形式需要考虑复数的实部和虚部。

考虑一个带有复系数的非齐次线性微分方程：

any(n)+an−1y(n−1)+⋯+a1y′+a0y=f(x)a_n y^{(n)} + a_{n-1} y^{(n-1)} + \cdots + a_1 y' + a_0 y = f(x) an​y(n)+an−1​y(n−1)+⋯+a1​y′+a0​y=f(x)

情况1：f(x)=eλxPm(x)f(x) = e^{\lambda x}P_m(x)f(x)=eλxPm​(x)

特解的形式为：

y∗=xkRm(x)eλxy^* = x^k R_m(x) e^{\lambda x} y∗=xkRm​(x)eλx

如果λ\lambdaλ为复数，例如λ=α+iβ\lambda = \alpha + i \betaλ=α+iβ，那么特解形式依旧成立，只是需要考虑λ\lambdaλ的复数部分。因此，

y∗=xkRm(x)e(α+iβ)x=xkRm(x)eαx(cos⁡(βx)+isin⁡(βx))y^* = x^k R_m(x) e^{(\alpha + i \beta)x} = x^k R_m(x) e^{\alpha x} (\cos(\beta x) + i \sin(\beta x)) y∗=xkRm​(x)e(α+iβ)x=xkRm​(x)eαx(cos(βx)+isin(βx))

情况2：f(x)=eλx[Pl(x)cos⁡(ωx)+Qn(x)sin⁡(ωx)]f(x) = e^{\lambda x} \left[P_l(x) \cos(\omega x) + Q_n(x) \sin(\omega x)\right]f(x)=eλx[Pl​(x)cos(ωx)+Qn​(x)sin(ωx)]

特解的形式为：

y∗=xkeλx[Rm(1)(x)cos⁡(ωx)+Rm(2)(x)sin⁡(ωx)]y^* = x^k e^{\lambda x} \left[R_m^{(1)}(x) \cos(\omega x) + R_m^{(2)}(x) \sin(\omega x)\right] y∗=xkeλx[Rm(1)​(x)cos(ωx)+Rm(2)​(x)sin(ωx)]

同理，若λ=α+iβ\lambda = \alpha + i \betaλ=α+iβ，特解可以写为：

y∗=xke(α+iβ)x[Rm(1)(x)cos⁡(ωx)+Rm(2)(x)sin⁡(ωx)]y^* = x^k e^{(\alpha + i \beta)x} \left[R_m^{(1)}(x) \cos(\omega x) + R_m^{(2)}(x) \sin(\omega x)\right] y∗=xke(α+iβ)x[Rm(1)​(x)cos(ωx)+Rm(2)​(x)sin(ωx)]

y∗=xkeαx(cos⁡(βx)+isin⁡(βx))[Rm(1)(x)cos⁡(ωx)+Rm(2)(x)sin⁡(ωx)]y^* = x^k e^{\alpha x} (\cos(\beta x) + i \sin(\beta x)) \left[R_m^{(1)}(x) \cos(\omega x) + R_m^{(2)}(x) \sin(\omega x)\right] y∗=xkeαx(cos(βx)+isin(βx))[Rm(1)​(x)cos(ωx)+Rm(2)​(x)sin(ωx)]

总结

待定系数法对于复系数的推广，特解形式仍然保持一致，只是需要在计算时处理复数的实部和虚部。具体步骤包括：

  1. 确定特解形式，保持与实数系数情况一致。
  2. 计算特解时，处理复数指数函数时的实部和虚部。
  3. 最终解通常会包含实部和虚部的组合。

通过这种方法，待定系数法可以顺利推广到复系数的情形。

非齐次线性偏微分方程

对于非齐次线性偏微分方程，待定系数法的推广会更复杂，但基本思想仍然适用。具体来说，我们需要处理多变量的情况，并且要考虑偏微分算子的特性。

假设我们有一个非齐次线性偏微分方程：

L[u]=f(x,y)L[u] = f(x, y) L[u]=f(x,y)

其中LLL是一个线性偏微分算子，u=u(x,y)u = u(x, y)u=u(x,y)是未知函数，f(x,y)f(x, y)f(x,y)是已知的非齐次项。

情况1：f(x,y)=eλx+μyPm(x,y)f(x, y) = e^{\lambda x + \mu y} P_m(x, y)f(x,y)=eλx+μyPm​(x,y)

对于这样的非齐次项，我们假设特解的形式为：

u∗=xkylRm(x,y)eλx+μyu^* = x^k y^l R_m(x, y) e^{\lambda x + \mu y} u∗=xkylRm​(x,y)eλx+μy

其中，Rm(x,y)R_m(x, y)Rm​(x,y)是xxx和yyy的多项式，kkk和lll是确定特解的常数。

情况2：f(x,y)=eλx+μy[Pl(x,y)cos⁡(ωx+νy)+Qn(x,y)sin⁡(ωx+νy)]f(x, y) = e^{\lambda x + \mu y} \left[ P_l(x, y) \cos(\omega x + \nu y) + Q_n(x, y) \sin(\omega x + \nu y) \right]f(x,y)=eλx+μy[Pl​(x,y)cos(ωx+νy)+Qn​(x,y)sin(ωx+νy)]

对于这样的非齐次项，我们假设特解的形式为：

u∗=xkyleλx+μy[Rl(1)(x,y)cos⁡(ωx+νy)+Rl(2)(x,y)sin⁡(ωx+νy)]u^* = x^k y^l e^{\lambda x + \mu y} \left[ R_l^{(1)}(x, y) \cos(\omega x + \nu y) + R_l^{(2)}(x, y) \sin(\omega x + \nu y) \right] u∗=xkyleλx+μy[Rl(1)​(x,y)cos(ωx+νy)+Rl(2)​(x,y)sin(ωx+νy)]

具体步骤

  1. 假设特解的形式：根据非齐次项f(x,y)f(x, y)f(x,y)的形式，假设特解u∗u^*u∗的形式。形式中包含指数函数、多项式和三角函数的组合。

  2. 代入方程：将假设的特解代入原偏微分方程L[u]=f(x,y)L[u] = f(x, y)L[u]=f(x,y)。

  3. 确定待定系数：通过比较方程两边的系数，确定假设特解中的未知常数kkk, lll和多项式Rm(x,y)R_m(x, y)Rm​(x,y)或Rl(1)(x,y)R_l^{(1)}(x, y)Rl(1)​(x,y)、Rl(2)(x,y)R_l^{(2)}(x, y)Rl(2)​(x,y)的系数。

示例

考虑以下偏微分方程：

∂2u∂x2+∂2u∂y2=eλx+μy\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = e^{\lambda x + \mu y} ∂x2∂2u​+∂y2∂2u​=eλx+μy

我们假设特解的形式为：

u∗=eλx+μyu^* = e^{\lambda x + \mu y} u∗=eλx+μy

代入偏微分方程：

∂2∂x2(eλx+μy)+∂2∂y2(eλx+μy)=λ2eλx+μy+μ2eλx+μy\frac{\partial^2}{\partial x^2} \left( e^{\lambda x + \mu y} \right) + \frac{\partial^2}{\partial y^2} \left( e^{\lambda x + \mu y} \right) = \lambda^2 e^{\lambda x + \mu y} + \mu^2 e^{\lambda x + \mu y} ∂x2∂2​(eλx+μy)+∂y2∂2​(eλx+μy)=λ2eλx+μy+μ2eλx+μy

显然，λ2+μ2=1\lambda^2 + \mu^2 = 1λ2+μ2=1。

对于更复杂的非齐次项，比如eλx+μy[Pl(x,y)cos⁡(ωx+νy)+Qn(x,y)sin⁡(ωx+νy)]e^{\lambda x + \mu y} \left[ P_l(x, y) \cos(\omega x + \nu y) + Q_n(x, y) \sin(\omega x + \nu y) \right]eλx+μy[Pl​(x,y)cos(ωx+νy)+Qn​(x,y)sin(ωx+νy)]，特解的假设形式和处理方法类似于常系数非齐次线性微分方程，只是要处理多变量和偏微分运算。

总结

待定系数法推广到非齐次线性偏微分方程时，需要考虑多变量的情况和偏微分算子的特性。基本步骤与常微分方程类似，但在处理具体解时会更复杂，需要考虑多项式和指数函数的组合及其在多变量下的表现形式。

实际情况中非齐次项f(x,y)f(x,y)f(x,y)不会太复杂, 可以观察f(x,y)f(x,y)f(x,y)形式直接写

  1. 观察是指数型还是圆函数型
  2. 观察多项式能否分离变量, 若可以分离, 则直接设y∗=g(x)h(y)y^*=g(x)h(y)y∗=g(x)h(y)
